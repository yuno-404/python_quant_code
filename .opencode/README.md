# OpenCode Local Rules

This folder stores local OpenCode guidance for this repository.

## What is tracked
- `skills/*.md` files in this directory are tracked intentionally.
- These files document repeatable operational rules for this repo.

## What is not tracked
- Other OpenCode runtime/cache/session files stay ignored by `.gitignore`.

## Mandatory rule
Before any push operation, read and follow `PUSH_WORKFLOW.md`.
