# Python Quant Code

Script-first quantitative finance learning repository.

## Public-facing structure
- `scripts/asset_price_modeling/`
- `scripts/stochastic_process/`
- `notebooks/asset_price_modeling/`
- `notebooks/stochastic_process/`

## Quick start
```bash
python -m venv .venv
. .venv/Scripts/activate
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```

## Run examples
```bash
python scripts/asset_price_modeling/symmetric_random_walk.py
python scripts/stochastic_process/stochastic_process_discrete_correlation.py
```
